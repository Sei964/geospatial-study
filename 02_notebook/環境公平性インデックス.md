# **第1章：環境公平性インデックス（EI）とは**

1.1 定義
**環境公平性インデックス（Environmental Equity Index, EI）** とは、 地域ごとの環境的な負荷と社会的な脆弱性を総合的に評価し、 **環境の不公平さを可視化する指標**である。

第2章：背景と目的
2.1 なぜ必要なのか
都市や地域では、以下のような**環境的不平等**が存在することがある：

- 空気の汚れがひどい地域に、低所得世帯が集中している

- 緑地や公園が少ない地域に、子どもや高齢者が多く住んでいる

- 災害リスクの高い場所に、社会的支援が届きにくい人々が暮らしている

こうした不公平を把握し、**誰もが健康で安全に暮らせる社会を実現するため**に、EIは活用される。

第3章：構成要素
3.1 環境指標

- 大気汚染（NO₂、PM2.5 など）

- 気温（ヒートアイランド現象）

- 騒音、振動

- 緑被率、自然環境の質

3.2 社会的脆弱性指標

- 所得水準

- 高齢者・子どもの割合

- 障がい者人口

- 外国人比率

- 教育水準や住宅環境

第4章：評価方法

4.1 データの標準化
各指標は単位やスケールが異なるため、**Zスコアなどで標準化**を行う。

4.2 指標の合成
標準化された環境指標と社会指標を加重平均などで合成し、 地域ごとのEIスコアを算出する。

4.3 空間単位
評価は、町丁目、メッシュ、行政区などの**空間単位**で行われる。

第5章：活用例
高リスク地域の特定と優先的な対策

環境政策や都市計画への反映

公平な資源配分のための意思決定支援

市民参加型のまちづくりの基盤

第6章：まとめ
環境公平性インデックスは、 **環境と社会の両面から地域の課題を見つけ出すための重要なツール**である。 持続可能で包摂的な社会を目指す上で、EIの活用は今後ますます重要になる。
